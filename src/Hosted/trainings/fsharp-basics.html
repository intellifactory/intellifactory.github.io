What is F# and why it matters

F# is a functional-first, multi-paradigm programming language which is becoming increasingly popular in finance, science, web and mobile development, data mining, machine learning, and more.


The language allows for better organized, reliable, reviewable, maintainable code compared to what a purely object-oriented approach can provide.


With us, you can quickly learn to use F# in real-world, complex projects.

How F# increases productivity

You can focus on how to write "next generation" applications using F#'s more advanced features.


F#'s advanced concepts enable programming sophistication that would be significantly harder in conventional programming languages.


  • Easier Concurrency, Asynchronous Programming, Parallel Programming
  • Ability to do compilation, interpretation, and transformations via F#'s Quotation facility.
  • Continuations
