Reactive programming

Functional reactive programming is an alternative to event-based approach for creating interactive applications.


With events, you have think about what the full effect of every user action possible would be.


With reactive programming, you can think about how parts of the application react to changes in other parts, including changes initiated by the user.


This leads to a more manageable application logic, enabling better user experience in the same or less development time.
